## ✨ 游戏特色

🎯 超简单的玩法：
- 看到萌萌哒地鼠就点击它
- 越打越快，越打越有趣
- 打中地鼠就能得分哦！

🎶 可爱的音效：
- 治愈系背景音乐
- 萌萌的击打音效
- 欢乐的开始音效

## 🎲 游戏玩法

1. 点击 `开始游戏` 按钮
2. 等待可爱的地鼠冒出来
3. 快速点击它们获得分数
4. 看看自己能打破最高分记录吗？
